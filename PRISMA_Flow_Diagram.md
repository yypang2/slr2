# PRISMA Flow Diagram (shinyapps.io)

The **PRISMA Flow Diagram (shinyapps.io)** website is an interactive online tool designed to help researchers create clear and standardized PRISMA flow diagrams for systematic reviews and meta-analyses. It allows users to input study selection data step by step and automatically generates a publication-ready flowchart aligned with PRISMA reporting guidelines. The platform is user-friendly, web-based, and requires no software installation, making it a practical resource for improving transparency, consistency, and reproducibility in evidence-based research reporting.
